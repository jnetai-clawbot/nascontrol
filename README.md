# NASControl - Android NAS Controller

Manage your home server / NAS from your Android device.

## Features
- System resource monitoring (CPU, RAM, disk, network)
- Disk health monitoring
- Docker container management
- Media streaming controls
- File explorer for NAS
- Backup management
- Server status dashboard
- Alert notifications

## Tech Stack
- Kotlin 1.9.22, Jetpack Compose, Material Design 3
- Room Database, KSP
- ARM64 only, MinSdk 29, TargetSdk 34
